Mapnik

Documentation Mapnik¶

Mapnik est une boîte à outils multi-plateformes en C++/Python pour développer des applications cartographiques. Mapnik est un logiciel libre et il est disponible sous licence LGPL (Lesser General Public Licence). Le coeur est constitué d’une bibliothèque partagée en C++ qui fournit des algorithmes / modèles pour l’accès à des données spatiales et pour la visualisation. Il peut être utilisé aisément à la fois pour du développement web comme bureautique permettant un développement rapide basé sur les méthodes agiles.
